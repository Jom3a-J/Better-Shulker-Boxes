 Better Shulker Boxes - Quick Start Guide

Interact with Shulker Boxes and Ender Chests directly from your inventory



 🎮 Essential Controls

Hover over a container in your inventory and use these shortcuts

 Action  Control  Description 
 
 Insert / Extract (Stacks)  `Left-Drag` / `Right-Drag`  Drag items into container (left) / extract to inventory (right) 
 Insert / Extract (Single) `Ctrl + Left-Drag` / `Ctrl + Right-Drag`  Precision mode inserts (left) or extracts (right) exactly 1 item 
 Sweep Insert / Extract `E`  Moves matching items into container / extracts from selected slot 
 Restock / Deposit  `R` / `Shift + R`  Refills matching hotbar items (R) / deposits matching items (Shift+R) 
 Cycle Slots  `Scroll Wheel` / `Arrows`  Navigates the tooltip preview grid 
 Paint Slots `Space + Drag`  Highlights multiple slots for visual actions 
 Sort Container  `G`  Cycles sort modes (Original Name Count Category) 
 Wireless Ender Chest `O`  Access Ender Chest anywhere (requires chest in inventory) 
 Dye Shulker Box  `Right-Click Dye`  Colors Shulker boxes directly in inventory 
 Open Settings  `B`  Opens config screen 


 🔧 Configurable Options (`B` Key)

* **Tooltip Preview** Show/hide the container contents preview grid
* **Precision Mode** Toggle the `Ctrl` key modifier for single-item actions
* **Fill Indicator** Display a stack-density level bar under item counts
* **Secondary Scroll** Toggle mouse wheel slot navigation
* **Alt-Force Tooltip** Hold `Alt` to keep tooltips on cursor when moving items
* **Compact Tooltip** Hide empty slots to save space (Hold `V` to expand)

 💡 Quick Tips

* **Wireless Access** You must carry an Ender Chest item in your inventory for remote access to work
* **Server-Safe** All actions are validated server-side to prevent items from duplicating
* **Mod Compatibility** Works completely on both singleplayer and multiplayer servers
